# Roteiro curto de apresentação

Bom dia/boa tarde. Nosso projeto se chama KRONOS/LINUX. Ele é um dashboard web de monitoramento Linux criado para demonstrar conceitos de Sistemas Operacionais e DevOps.

O sistema apresenta informações operacionais como uso de CPU, memória RAM, armazenamento em disco, processos ativos, containers e logs do sistema. A ideia é simular um painel usado por equipes de infraestrutura para acompanhar o estado de um servidor Linux.

A parte principal do projeto está ligada a Sistemas Operacionais Linux. Utilizamos scripts Shell para monitoramento, healthcheck, backup e limpeza de logs. Também demonstramos conceitos como processos, logs, terminal, systemd, permissões e automação de tarefas administrativas.

Na parte DevOps, o projeto foi organizado com Git/Git Flow, possui testes automatizados, pipeline CI/CD com GitHub Actions, Dockerfile para containerização e arquivos Kubernetes para orquestração.

Com isso, o projeto integra desenvolvimento, infraestrutura e operação, mostrando como uma aplicação pode ser executada, validada, automatizada e monitorada em um ambiente Linux.
