# Entrega do Projeto

## Nome do projeto
KRONOS/LINUX — Dashboard de Monitoramento Linux e DevOps

## Tipo
Aplicação web com foco em monitoramento operacional Linux.

## Tecnologias
- HTML, CSS e Node.js
- Shell Script
- Git e Git Flow
- GitHub Actions
- Docker
- Kubernetes

## Pontos principais
- Dashboard visual de infraestrutura Linux
- Monitoramento de CPU, RAM, disco, processos, containers e logs
- Scripts Linux para automação
- Testes automatizados
- Pipeline CI/CD
- Containerização com Docker
- Estrutura Kubernetes
